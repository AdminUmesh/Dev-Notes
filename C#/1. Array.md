# C# Arrays

### What is an Array?

An array is a collection of elements of the same type stored in a single
variable.

### Declaration

``` csharp
int[] arr = { 10, 20, 30, 40 };

// or
int[] arr2 = new int[5];
```

### Access Elements

``` csharp
Console.WriteLine(arr[0]); // 10
```

### Loop Through Array

``` csharp
foreach (var item in arr)
{
    Console.WriteLine(item);
}
```
------------------------------------------------------------------------
## **Multi-Dimensional Array (Jagged Array)**

``` csharp
int[][] myNumbers = new int[][]
{
    new int[] {1, 2, 3, 4},
    new int[] {5, 6, 7}
};

Console.WriteLine(myNumbers[1][2]); // 7
```

### Loop Through Multi-Dimensional Array

``` csharp
for (int i = 0; i < myNumbers.Length; i++)
{
    for (int j = 0; j < myNumbers[i].Length; j++)
    {
        Console.WriteLine(myNumbers[i][j]);
    }
}
```

# **Built-in Array Methods**

### Array.Reverse()

``` csharp
int[] arr = { 1, 2, 3 };
Array.Reverse(arr);
```

### Array.Sort()

``` csharp
int[] arr = { 5, 2, 8, 1 };
Array.Sort(arr);
```

### Array.IndexOf()

``` csharp
int[] arr = { 10, 20, 30 };
int index = Array.IndexOf(arr, 20); // 1
```

### Array.Copy()

``` csharp
int[] source = { 1, 2, 3 };
int[] dest = new int[3];

Array.Copy(source, dest, 3);
```

### Array.Clear()

``` csharp
int[] arr = { 1, 2, 3 };
Array.Clear(arr, 0, arr.Length);
```

### Array.Exists()

``` csharp
int[] arr = { 1, 2, 3 };

bool exists = Array.Exists(arr, x => x == 2);
```

### Array.Find()

``` csharp
int[] arr = { 1, 2, 3 };

int result = Array.Find(arr, x => x > 1);
```

### Array.FindAll()

``` csharp
int[] arr = { 1, 2, 3, 4 };

int[] result = Array.FindAll(arr, x => x % 2 == 0);
```

### Array.Resize()

``` csharp
int[] arr = { 1, 2, 3 };

Array.Resize(ref arr, 5);
```

### Array.BinarySearch()

``` csharp
int[] arr = { 1, 2, 3, 4, 5 };

int index = Array.BinarySearch(arr, 3);
```

## **Convert**
**1. Convert char[] → string**
```c#
char[] arr = { 'H', 'e', 'l', 'l', 'o' };
string result = new string(arr);
Console.WriteLine(result); // Hello
```

**2. Convert string → char[]**

```c#
string str = "Hello";
char[] arr = str.ToCharArray();
Console.WriteLine(arr[0]); // H
```

**3. Convert int[] → string**
👉 Using string.Join()

```c#
int[] numbers = { 1, 2, 3, 4 };
string result = string.Join(",", numbers);
Console.WriteLine(result); // 1,2,3,4
```

**4. Convert string → int[]**
👉 Split + Parse

```c#
string str = "1,2,3,4";
int[] arr = str
    .Split(',')
    .Select(int.Parse)
    .ToArray();
```
⚠️ Requires: using System.Linq;

**5. Convert string[] → string**
```c#

string[] arr = { "Hello", "World" };
string result = string.Join(" ", arr);
Console.WriteLine(result); // Hello World
```

**6. Convert string → string[]**

```c#
string str = "Hello World";
string[] arr = str.Split(' ');
Console.WriteLine(arr[0]); // Hello
```

**7. Convert int[] → char[] (numbers to characters)**

```c#
int[] nums = { 65, 66, 67 };
char[] chars = nums.Select(x => (char)x).ToArray();
Console.WriteLine(new string(chars)); // ABC
```

**8. Convert char[] → int[]**

```c#
char[] chars = { '1', '2', '3' };
int[] nums = chars.Select(c => int.Parse(c.ToString())).ToArray();
```
