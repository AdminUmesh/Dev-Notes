# Character
char (short for character) is a data type used to represent a single character `(letters, digits, punctuation marks, whitespace characters and special characters like newline characters)`.

**Creating a Character object:**
- char ch='a';

## **Methods with `char`**
The Character class is immutable like String class i.e once it’s object is created, it cannot be changed.
```csharp
Character.isDigit('A') //return true if char is a number
Character.isLetter('A') //return true if char is a Letter
Character.isWhitespace('A') //return true if char is a whiteSpace
Character.isUpperCase('A') //return true if char is in uppercase
Character.isLowerCase('A') //return true if char is in lowercse
Character.toUpperCase('a') //return char in uppercase
Character.toLowerCase('A') //return char in lowercase
Character.toString('x') //return char in string
```

## **ASCII value** (American Standard Code for Information Interchange)
ASCII is a standard that assigns a numeric value (code) to each character and symbol.

- Every character (A, B, 1, @, etc.) has a unique number
- Computers store and process characters using these numbers

### **There are 3 ways to print ASCII value or code of a specific character**

**1. Using brute force Method**

 `simply assign the character to a new variable of integer type.`
```c#
using System;

class Program
{
    static void Main(string[] args)
    {
        char ch = 'A';
        int ascii = (int)ch;
        Console.WriteLine("The ASCII value is " + ascii);
    }
}

```

**2. Using the type-casting Method**

`Type casting in C# is the process of converting a value from one data type to another.`
```c#
using System;

class Program
{
    static void Main(string[] args)
    {
        char ch = 'A';
        Console.WriteLine("The ASCII value is " + (int)ch);
    }
}
```
 **3. Using the format specifier Method**

`A format specifier is used to display a value in a specific format during output.`

```C#
using System;

class Program
{
    static void Main(string[] args)
    {
        char character = 'A';
        string formatSpecifier = $"{(int)character}";
        Console.WriteLine("The ASCII value is " + formatSpecifier);
    }
}

```
