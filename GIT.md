---------------------------------------------------------------------------------------------------------------
|                                                     **GIT**                                                 |
---------------------------------------------------------------------------------------------------------------

**Check git Version**
>>> git --version

**Check git Status**
>>> git status

**Check git Branch**
>>> git branch

**Change git Branch**
>>> git checkout 'another_branch_name'
>>> git checkout origin 'another_branch_name'

**Add file Individual**
>>> git add filename

**Add All file**
>>> git add .

**GIT Commit**
>>> git commit -m "enter your message"

**GIT Push**
>>> git push
>>> git push origin branch_name
