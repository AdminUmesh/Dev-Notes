## What is a Callback Function?

A callback function is a function that is passed as an argument to
another function and is executed later.

------------------------------------------------------------------------

## Basic Example

``` js
function greet(name, callback) {
    console.log("Hello " + name);
    callback();
}

function sayBye() {
    console.log("Goodbye!");
}

greet("Umesh", sayBye);
```

------------------------------------------------------------------------

## Output

    Hello Umesh
    Goodbye!

------------------------------------------------------------------------

## Why Callbacks?

-   To handle asynchronous operations
-   To execute code after a task completes
-   Used in events, timers, API calls

------------------------------------------------------------------------

## Example with setTimeout

``` js
setTimeout(function() {
    console.log("Executed after 2 seconds");
}, 2000);
```

------------------------------------------------------------------------

## Asynchronous Callback Example

``` js
function fetchData(callback) {
    setTimeout(() => {
        callback("Data received");
    }, 2000);
}

fetchData(function(data) {
    console.log(data);
});
```

------------------------------------------------------------------------

## Callback Hell (Problem)

``` js
getUser(function(user) {
    getOrders(user, function(orders) {
        getPayment(orders, function(payment) {
            console.log(payment);
        });
    });
});
```

------------------------------------------------------------------------

## Problems with Callbacks

-   Hard to read (nested)
-   Difficult error handling
-   Hard to maintain

------------------------------------------------------------------------

## Solution

-   Promises
-   Async/Await

------------------------------------------------------------------------

## Key Points

-   Callbacks are functions passed as arguments
-   Used for async operations
-   Can lead to callback hell
-   Replaced by Promises in modern JS

------------------------------------------------------------------------

## One-Line Summary

A callback is a function passed into another function to be executed
later.
