
# Windows Installation Guide (Using Media Creation Tool)

## Requirements
- Windows PC/Laptop
- Internet connection
- USB Pendrive (8 GB or more)
- Backup of important data

---

## Step 1: Download Media Creation Tool
1. Go to https://www.microsoft.com/software-download/windows11
2. Download **Windows 11 Media Creation Tool**
3. Run the tool as **Administrator**

---

## Step 2: Create Bootable USB
1. Insert USB pendrive
2. Open Media Creation Tool
3. Accept license terms
4. Select **Create installation media (USB flash drive)**
5. Choose:
   - Language
   - Edition: Windows 11
   - Architecture: 64-bit
6. Select **USB flash drive**
7. Choose your USB device
8. Wait until download & creation completes

## Step 3: Boot from USB
1. Insert USB into target laptop
2. Turn on laptop
3. Press **F12 (Dell)** or **F9/F10/F11 (other brands)**
4. Select **UEFI: USB Storage Device**
